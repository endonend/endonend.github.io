I added a few more Micro.blog shortcuts to my MarsEdit setup. To add these, open up the New Post window, then under the Format dropdown select Customize. Here you can add new macro entries. Add the code I shared below to the Opening Markup box for each macro.

### Easily link to another Micro.blog user

Here are two ways to easily link to another Micro.blog user (useful for sending webmentions):

<ul>
<li>This option will prompt you to enter the username:</li>
</ul>

<img src="http://endonend.micro.blog/uploads/2018/77b65d2b99.jpg" alt="Mb enter" title="mb-enter.png" />

<p>Copy the macro:
<pre>[@#asktext Enter Text:#](https://micro.blog/#asktext Enter Text:#)</pre></p>

<ul>
<li>If you copy the username to your clipboard this option will grab it and automatically enter:</li>
</ul>

<img src="http://endonend.micro.blog/uploads/2018/028ce3f8d5.jpg" alt="Mb paste" title="mb-paste.png" />

Copy the macro:
<pre>[@#pastetext#]([micro.blog](https://micro.blog/#pastetext)#)</pre>
